# Code Quality Analyzer

Вебзастосунок для швидкого аналізу якості коду. Дозволяє завантажувати файли, отримувати метрики та ризики, переглядати підсумок аналізу і системні логи в одному інтерфейсі.

## Можливості
- Завантаження файлів для аналізу
- Дашборд метрик і ризиків
- Підсумкові рекомендації та статуси
- Журнал подій/логів системи

## Технології
- React + TypeScript
- Vite
- Vitest

## Як запустити локально
1. Встановіть залежності: `npm install`
2. Запустіть dev-сервер: `npm run dev`
3. Відкрийте у браузері адресу з терміналу

## Структура
- `src/pages` — сторінки застосунку
- `src/widgets` — великі UI-блоки
- `src/entities` — доменні сутності
- `src/shared` — спільні компоненти та утиліти
